✨ 核心功能

简洁而不简单

每一个功能都经过精心打磨，让日常任务管理变得顺手自然

🗓️

月视图日历

7 列网格布局，支持周一/周日起始切换。农历、节气、法定节假日全月显示（含跨月日期），调休标注一目了然。双击顶部年月标题即可弹出滚动选择器，快速跳转任意年月。鼠标滚轮逐周翻页，每月1号自动标注月份标识。

✅

任务管理

双击日期格即可添加任务，支持浮窗编辑、标题列表、颜色标记。7 种预设背景色标记重要日期，右键菜单快速操作。

🎨

外观定制

5 种深色主题，30%~100% 透明度滑块，背景色字体色独立可调，字体大小 10~18px，打造你的专属桌面。

🖥️

桌面挂件

固定窗口不可拖动，像桌面小部件常驻前台。支持窗口置顶、大小可调、开机自启动（路径变更自动同步）。

☁️

云同步

支持 WebDAV / 自建服务器云端同步任务数据，右上角 ☁ 按钮一键操作，多设备数据不丢失。

📊

数据管理

JSON 导出/导入备份恢复，Excel xlsx 格式导出，兼容旧版 v3 SQLite 数据格式导入。

🌍

国际化

中文/英文双语切换，全界面翻译适配：菜单、弹窗、提示、日志均完整翻译。

📦

自动更新

启动后自动检查 GitHub 新版本，设置中可开关，始终保持最新。

🤖 AI 对接

MCP Server
让 AI 读写你的任务

内置 MCP Server，AI 助手（如 WorkBuddy）可直接操作你的任务清单。 设置 → 高级 → 配置 MCP，一键生成连接器配置代码，零门槛接入。

list_tasks
add_task
complete_task
delete_task
export_all_tasks
batch_add_tasks
// WorkBuddy MCP 配置
{
  "TodoList": {
    "command": "node",
    "args": ["mcp-server/index.js"],
    "cwd": "应用安装目录"
  }
}
